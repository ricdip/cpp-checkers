# cpp-checkers

A C++ implementation of the checkers game using also CMake and GoogleTest for learning purposes.

## Build debug

### Build main
```bash
user@host:~$ make build_debug
```

### Build tests
```bash
user@host:~$ make build_debug_test
```

## Run debug

### Run main
```bash
user@host:~$ make run
```

### Run tests
```bash
user@host:~$ make run_tests
```
